AVL Trees: Formal Definition

  • Balance factor for a tree node n :
    • Height of n's right subtree minus height of n's left subtree $$BF_n = Height_{n.right} - Height_{n.left}$$
    • Reminder: Height of empty tree is -1
  • A binary search tree is an AVL tree if:
    • balance factor of each node is 0, 1, or -1 (if no node's two child subtrees differ in height by more than 1)
