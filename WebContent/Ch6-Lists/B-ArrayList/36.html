Adding elements

  • Elements are added dynamically to the list:
    • ArrayList list = new ArrayList();
System.out.println("list = " + list);
list.add("Tool");
System.out.println("list = " + list);
list.add("Phish");
System.out.println("list = " + list);
list.add("Pink Floyd");
System.out.println("list = " + list);
  • Output:
    • list = []
list = [Tool]
list = [Tool, Phish]
list = [Tool, Phish, Pink Floyd]
