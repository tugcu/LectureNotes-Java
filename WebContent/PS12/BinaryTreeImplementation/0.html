Node Class

  • In binary trees, data is stored in simple objects, called Node.
  • Each node can have up to 2 children, whose pointers are stored in the parent node.
  • Overall, below is the implementation of a generic Node object: 
public class Node<E> {
	E data;
	Node<E> leftChild;
	Node<E> rightChild;

	public Node(E data) {
		this.data = data;
	}
}
