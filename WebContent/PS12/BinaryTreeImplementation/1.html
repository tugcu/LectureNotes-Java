BinaryTree Class

  • Just like linked lists, binary trees can be traversed recursively. It is almost always sufficient to store just the root of the tree.
  • Many operations can be implemented recursively. Think about the infix traversal of a binary tree: For each node, the left child and its descendant are processed first. This is followed by the node itself and the right child and its descendant are processed at the end. So, why not to implement a method that takes a Node as the parameter and call it recursively? 
				
public class BinaryTree<T> {
	Node<T> root;
	
	public void infixTraverse() {
		infixTraverse(root);
	}
	
	private void infixTraverse(Node<T> root) {
		if (root != null) {
			System.out.println(root.data);
			infixTraverse(root.leftChild);
			infixTraverse(root.rightChild);
		}
	}
}
