Chapter 13.4 - gets & puts

gets(char S[]):
  • Gets a string of characters terminated by a newline from the standard input stream, keyboard, and puts in string S.
  • It replaces the newline with a null character in S.
  • This function is defined in stdio.h library. You must ensure that S has enough capacity.
puts(char S[]):
  • Copies the null terminated string S to the standard output stream, monitor, and appends a newline character.
  • It is defined in stdio.h library.
