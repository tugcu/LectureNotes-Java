Chapter 13.5 - atoi & atof

  • Converts a string S to its corresponding integer.
  • If the string cannot be converted to an integer it returns zero.
  • This function is defined in stdio.h library.

atoi("345") returns an integer 345. 
atoi("abc") returns zero.
atoi("12ad3") returns 12.

double atof(char S[]):
Converts a string S to the corresponding floating number.
