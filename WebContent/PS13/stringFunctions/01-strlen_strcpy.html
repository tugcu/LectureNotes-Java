Chapter 13.1 - strlen & strcpy

strlen(char S[]): Returns the number of characters in S, not counting the null character.

strcpy(char destination[], char source[]): Copies the string source to the string destination. The function stops copying after the null character has been moved. You must ensure that the destination string has enough capacity.
