Chapter 13 - Strings

  • The string is a memory block whose last element is indicated by the null character, '\0'.
  • The null character provides us to know the end of the block. This feature is important since it is very common to use texts in our programs.
  • The memory block can be kept as a character array or its first address is kept in a character pointer.
  • Suppose we have an character pointer P, not a string. To print its content we must know its size and the code for this task is: 
    	
	for (i = 0; i < size; i++)
		printf("%c",P[i]);
  • If we use strings, we should set the last element of P to the null character. Therefore, we do not need to keep the value of the size of the block. Our code becomes: 
    	
	for (i = 0; P[i] != '\0'; i++)
		printf("%c",P[i]);
  • Actually many facilities for strings can be found in C. For example, printf function can be applied on strings directly, the conversion specifier for strins is %s. Our code becomes: 
    	
	printf("%s",P);
  • A string constant can be given in double quotes (remember the string given in double qutoes in printf function). The string constant "hello world" is an array of characters.
  • In internal representation, the array is terminated with the null character, so the program can find the end of the string.
  • The length of a string is the numbers of characters in a string plus 1, which is for the null character. Let us examine the declaration of strings: 
    	
	char a[] = "blue";
  • This declaration creates an array of size 5 and initializes the contents of array. This is exactly the same as the first statement of the following code. 
    	
char a[] = {'b','l','u','e','\0'};
a[0] = 'a';        /* a becomes "alue" */
a[2] = '+;         /* a becomes "al+e" */
a[5] = '+';        /* you should not try to access the sixth element, 
                      since this is not an allocated location */
a[4] = '*'         /* it is OK, but you lose the ending point  (null character)
                      information as the result of this statement */
