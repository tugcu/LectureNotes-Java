toString()

  • This method returns the string representation of an object.
  • By default, the hashcode value is printed, which is something like the signature of an instance.
  • If you print any object, java compiler internally invokes the toString() method on the object. So overriding the toString() method, returns the desired output, it can be the state of an object etc. depends on your implementation.
  • For more examples, see http://www.javatpoint.com/understanding-toString()-method
