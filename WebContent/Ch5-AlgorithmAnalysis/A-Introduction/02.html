Incentive for asymptotic analysis of an algorithm

  • Given two different algorithms for the same problem, how can I determine which one is better?
    • E.g.: Which accounting program to purchase?
      • Eventually, they all do the same thing, but how about the performance?
      • I can’t afford entering all company data using each program, just for the sake of performance testing.
    • How would my algorithm work when the problem size significantly increases?
      • E.g.: What would be the performance of the web search algorithm I have in my mind?
        • I can estimate it would work fine for a small web site, but how will it perform for the whole WWW, like Google Search?
        • I don’t want to see that it fails after I implement it.
