3.2. Floating point numbers

Floating point numbers are the real numbers, i.e., the numbers with fractional parts (e.g.: 1.54). Just like integers being input and output using %d, floats are input and output using %f.



Exercise:

Write a program that reads two floats from the user and calculates the sum.



Another method for declaring a real number is using double, which can store real numbers with higher precision.
%lf is used for reading and writing doubles.



Exercise:

Write a program that reads two doubles from the user and calculates the sum.
