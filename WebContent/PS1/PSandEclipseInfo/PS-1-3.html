Views

  • In Eclipse, all the tiny windows you see are views:
    • Package Explorer
    • Console
    • SoftDev4U Main
    • SoftDev4U Browser
    • Problems
    • ...
  • Do not get scared if you close windows. Just look at the menu bar: Window->Show View
  • Click "Other" for more views
  • There is also what is called perspective, which is a set of windows opened for your task. For different tasks such as coding, debugging, versioning, etc, there are different perspectives.
