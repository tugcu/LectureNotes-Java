Brief Introduction to Eclipse

  • Eclipse is an integrated development environment (IDE) mainly for Java and other languages such as C++, Prolog, PHP, etc.
  • In this course, we will only write in Java.
  • Below are what we will do in Eclipse. You are expected to be familiar with all concepts:
    1. Creating new projects
    2. Importing projects
    3. Running/debugging
    4. Writing Javadoc
    5. Writing/running JUnit test cases
  • Eclipse is easy to use. Just look around the menu bar, menu items and buttons.
