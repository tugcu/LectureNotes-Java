Method signatures

  • Fully describe the signature for each method
  • The signatures distinguish the methods from each other
    • The signature includes the number, order, and types of the parameters
  • Use a @param tag to describe each parameter
    • @param tags should be in the correct order
    • Don’t mention the parameter type; Javadoc does that for you
    • Use @return tag to describe the result (unless it’s void)
