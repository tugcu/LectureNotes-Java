PS 15 - Struct

    We used arrays to define sets of multiple variables of the same type.

    Similarly, we use structs to define sets of variables of different types.

    Structs are defined before the main function. A simple struct definition is: 
    
struct Student{
	int id;
	int midterm1Grade;
	int midterm2Grade;
	int finalGrade;
	char letterGrade;
};
    Here, Student becomes a bundle of variables. Just like we create variables of different types in the program, we can create variables of a defined struct type like: 
    
struct Student s1;
    Elements of a struct variable are accessed with . (dot) operator and used just like any other variable. i.e: 
    

#include <stdio.h>
#include <stdlib.h>
#include <teachingcodes.h>

struct Student{
	int id;
	int midterm1Grade;
	int midterm2Grade;
	int finalGrade;
	float overallGradeOverFour;
	char letterGrade;
};

int main() {
	TC_BEGIN;

	//create two students here
	struct Student s1;
	struct Student s2;

	//set the id's of them
	s1.id = 2014800042;
	s2.id = 2011700321;

	//read their grades here
	scanf("%d", &s1.midterm1Grade);
	scanf("%d", &s2.midterm1Grade);
	scanf("%d", &s1.midterm2Grade);
	scanf("%d", &s2.midterm2Grade);
	scanf("%d", &s1.finalGrade);
	scanf("%d", &s2.finalGrade);

	//calculate their overall grades (over 4.0)
	s1.overallGradeOverFour = (s1.midterm1Grade*0.3f
			+ s1.midterm2Grade*0.3f
			+ s1.finalGrade*0.4f)/25.0f;
	s2.overallGradeOverFour = (s2.midterm1Grade*0.3f
				+ s2.midterm2Grade*0.3f
				+ s2.finalGrade*0.4f)/25.0f;

	//calculate their letter grades
	s1.letterGrade = 'A'+(4-(int)s1.overallGradeOverFour);
	s2.letterGrade = 'A'+(4-(int)s2.overallGradeOverFour);

	//print the results
	printf("%d: %.2f (%c)\n",s1.id, s1.overallGradeOverFour, s1.letterGrade);
	printf("%d: %.2f (%c)\n",s2.id, s2.overallGradeOverFour, s2.letterGrade);

	TC_END;
	return 0;
}
    Elements of a struct can be another struct, or an array. Arrays of structs can also be created. For example: 
    


struct Class
{
	int classID;
	int studentCount;
	struct Student students[100];
	struct Student theBestStudent;
	float curveGrade;
};
    An instance of a struct can be assigned to another one. For example: 
    

struct Class c;
struct Student theBestStudent;
// .......
// .......
theBestStudent = c.students[2];
//...........
//............
struct Student s2;
struct Student s3;
//...........
//............
s3 = s2;
    When such an assignment is done, all the values of the elements of s2 is copied into the corresponding elements of s3.
