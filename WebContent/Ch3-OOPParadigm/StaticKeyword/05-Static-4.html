Use of static Keyword

  • When a number of objects are created from the same class, each instance has its own copy of class variables. But this is not the case when a field is declared as static.
  • A static method or a static variable is not attached to a particular object, but rather to the class as a whole.
  • A static variable is a variable who's single copy in memory is shared by all objects of that class, so any modifications to the static variable will modify its value in all objects.
