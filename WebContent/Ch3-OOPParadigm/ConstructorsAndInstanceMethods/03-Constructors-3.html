Multiple Constructors

  • You can have several constructors with differentiating constructors 
    public class MyClass {
	MyClass() { ……
	}
	MyClass(int a, String b) {  ...
	}
	MyClass(String x, int y) {  ...
	}
}

MyClass obj1 = new MyClass();
MyClass obj2 = new MyClass(2,”Two”);
MyClass obj3 = new MyClass(”Three”, 3);

  • ATTENTION: If you write alternative constructors (with parameters) but skip writing the default constructor, it means that the use of the default constructor is banned. 
    public class MyClass {
	/* Default constructor skipped */
	MyClass(int a, String b) {  ...
	}
	MyClass(String x, int y) {  ...
	}
}

MyClass obj1 = new MyClass();            /* ERROR */
MyClass obj2 = new MyClass(2,”Two”);     /* OK */
MyClass obj3 = new MyClass(”Three”, 3);  /* OK */
