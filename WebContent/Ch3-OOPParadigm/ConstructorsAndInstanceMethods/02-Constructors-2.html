Multiple Constructors

  • Syntax: 
    public class CLASSNAME {
	CLASSNAME ( ) {  ...
	}

	CLASSNAME ([ARGUMENTS]) {  ...
	}
}

CLASSNAME obj1 = new CLASSNAME();
CLASSNAME obj2 = new CLASSNAME([ARGUMENTS]);

  • Example: 
    public class Automobile {

    /*  Default constructor*/
    Automobile ( ) {  ...
    }

    /* Alternative constructor with one String parameter */
    Automobile (String plate) {  ...
    }
}

...

/* Triggers default constructor */
Automobile CarForUmay = new Automobile();

/* Triggers constructor with String parameter */
Automobile CarForBilge = new Automobile("34 XW 245");
