Getters and Setters

  • Note that the getter and setter methods may be more complex if necessary. For example:
    • You might want to return the translated version of the stored value (e.g., return "one", "un", "ein", or "bir" for the value 1 depending on the selected language)
    • You might want to set or return x,y, and z coordinates of a point in 3D-space together.
