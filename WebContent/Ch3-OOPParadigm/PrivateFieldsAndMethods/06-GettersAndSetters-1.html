Getters and Setters

  • You may provide controlled access to a private field by creating getter and/or setter methods.
    • Getter (Accessor) Method: A method that returns the value of a private field. Thus, it provides read-only access to the private field.
    • Setter (Mutator) Method: A method that sets the value of a private field to the parameter of the method. Thus, it provides write access to the private field.
  • The convention is to set the names of these methods to getVariableName and setVariableName, respectively.
