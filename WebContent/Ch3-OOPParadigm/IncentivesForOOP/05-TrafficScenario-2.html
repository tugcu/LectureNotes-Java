Example Scenario (via Object Oriented Programming)

  • Let's change our mind-set and think outside the box.
  • Why not write our program just like the way the real world functions!
  • If the real world functions as a collection objects that interact with each other to form the system, let's follow the same path.
    • Let each vehicle be an object
    • Let them store information about their own status (coordinates, speed, direction, destination, etc.) and also a set of actions (speed up, slow down, turn left, turn right, stop, etc.)
    • Just define how each a vehicle acts in each given state. Then, let the system run.
  • So, the traffic is the system composed of all of these vehicles somehow moving (hopefully in harmony).
