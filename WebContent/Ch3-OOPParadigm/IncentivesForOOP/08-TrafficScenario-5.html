Example Scenario (via Object Oriented Programming)

  • Note that bus, automobile, bicycle, etc. are special types of classes.
    • They have some features in common (such as ability to move, carry load/people, have an operator, etc.)
    • Shared features should belong to the common ancestor.
    • They also have unique features (such as tires, steel wheels, etc.)
    • Unique feature should belong to corresponding class
