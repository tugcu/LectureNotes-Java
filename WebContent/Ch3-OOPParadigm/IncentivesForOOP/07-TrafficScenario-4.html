Example Scenario (via Object Oriented Programming)

  • For the system to work properly, we need to define each class (automobile, truck, bus, motorcycle, bicycle) in detail.
  • However, the vehicles could be organized into several classes: automobiles, buses, and trucks.
  • Yet, we realize that these vehicles have many things in common. Why should we implement the similar components in each class?
  • Instead, if we benefit from abstraction by defining the classes in a hierarchy, we may put the common features higher in the hierarchy so that the descendants lower in the hierarchy inherit the common features from their parents
