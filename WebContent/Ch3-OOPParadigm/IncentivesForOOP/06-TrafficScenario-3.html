Example Scenario (via Object Oriented Programming)

  • We have one challenge: We have millions of objects to define.
  • However, the vehicles could be organized into several classes: automobile, bus, truck, motorcycle, and bicycle
  • So, rather than defining how each vehicle acts individually, let's just define how each class of vehicle acts.
  • Then, we can create several objects from each class
    • Thus, a class is a type whereas an object is a specific entity of that type.
    • For example, "automobile" is a class (type of vehicle) whereas a specific automobile with plate number "34 M 4102" is a specific object of "automobile" class.
    • We say "an object is an instance of its class".
