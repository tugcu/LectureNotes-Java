Classes vs. Objects

  • A class defines a type
    • E.g.: An integer stores whole numbers
    • E.g.: A sheep has a fluffy fur and gives milk
      • We just define what a sheep or integer is like. A sheep or an integer is not created by these definitions.
    • An object is a specific instance of a class
      • E.g.: count is an integer
      • E.g.: Dolly is a sheep
    • The class serves as a template for the object
    • You may have several objects (instances) of the same class
