Accessing Methods and Fields of an Object

  • However, if you try to access the fields or methods of an object from another object, you must do so through the object identifier.
public class ClassA {
	int a;
	
	public void m1() {
		a = 5;
	}
}

public class ClassB {
	ClassA x;
	
	public void k() {
		x.a = 5;
		x.m1();
	}
}
