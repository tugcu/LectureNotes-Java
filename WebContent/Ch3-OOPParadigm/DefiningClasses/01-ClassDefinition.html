Class Definition

  • Up to now, you have written Java programs composed of a single file (which also includes the main method).
  • From now on, you will be writing programs composed of multiple files.
    • Class names should be capitalized.
    • Each class should be written in a separate file.
      • (Ignore inner class concept)
      • The class name and the file name should be the same.
    • Only one of the files should have the main method.
