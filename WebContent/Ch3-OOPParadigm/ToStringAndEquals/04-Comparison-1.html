Comparing Objects

  • Since the variable keeps only the reference to the object, comparing two variables with the == operator does not compare the contents of the objects but their references.
  • Therefore, for the code segment 
    Course c1, c2;
c1 = new Course("CMPE160", "Intro to OOP");
c2 = new Course("CMPE160", "Intro to OOP");

if (c1 == c2)
	System.out.println("c1 and c2 are the same");
else
	System.out.println("c1 and c2 are the different");
    will show that they are different.

    Why?

    Because they are separate objects with different references.
