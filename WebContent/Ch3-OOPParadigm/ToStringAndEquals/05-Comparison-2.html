Comparing Objects

  • In order to compare two objects, we should use the equals method of the object rather than the == operator on the reference variables.
  • Then, we can implement the equals method as we wish, depending on "what equality means for us."
    • The meaning of "equality" might be quite different depending on the application.
    • Java cannot know how to decide on the equality of two objects you have defined.
    • For example, how can Java decide whether two courses are equivalent? Only the programmer can define it.
  • Actually, there is a default equals method, but it functions just like the == operator.
  • However, we can override its behaviour as we wish.
