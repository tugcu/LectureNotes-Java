JUnit

  • JUnit annotations:
    • @Before
    • @After
    • @BeforeClass
    • @AfterClass
  • Using the annotations, it is possible to define when the test methods will be executed.
  • To test the conditions, you can use the methods in the Assert class.
    • assertEquals()
    • assertTrue()
    • assertFalse()
    • assertNull()
    • assertNotNull()
    • assertSame()
    • assertNotSame()
